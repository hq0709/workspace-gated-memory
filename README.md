LaTeX source for the paper "Remember What You Thought, Not What You Said:
Workspace-Gated Episodic Memory for LLMs" (under review).

- `sections/` — paper sections (intro, related, setup, results, extensions, conclusion, appendix)

- `figures/` — all figures (PDF), auto-generated

All numbers are regenerated from raw JSON records by `pilot/make_paper_assets.py`
in the experiment repository.
